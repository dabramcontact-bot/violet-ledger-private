# Violet Ledger

Рабочая система закупок со структурой:

**Обзор → Запросы → PI → Логистика → Платежи**

## Возможности

- самостоятельные реестры запросов, PI, логистики и платежей;
- связи `Запрос → PI → Логистика / Платежи`;
- три статуса запроса: «Запрос отправлен», «Предложение получено», «Внесено в просчёт»;
- полный цикл PI с отдельными отметками писем на код ТН ВЭД и создание номенклатуры;
- многотоварная логистика и распределение стоимости поровну, по количеству, стоимости или вручную;
- автоматический расчёт стоимости логистики на единицу;
- загрузка и скачивание документов через Supabase Storage;
- поиск, фильтры и отдельный Excel-экспорт во всех рабочих разделах;
- адаптивный светлый premium-интерфейс;
- роли `admin`, `editor`, `viewer` и Row Level Security.

## Обновление Supabase

Перед публикацией текущей версии один раз выполните файл
`supabase/violet-ledger-upgrade.sql` в **Supabase → SQL Editor → New query**.

Миграция:

- создаёт отдельную таблицу PI;
- обновляет запросы, логистику и платежи;
- переносит прежние PI-этапы из запросов;
- удаляет устаревший операционный маршрут;
- создаёт журнал активности и приватное файловое хранилище;
- добавляет связи, индексы, проверки и RLS-политики.
